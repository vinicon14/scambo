# i18n: Scambo em 20 Idiomas — Plano de Implementação

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Internacionalizar a plataforma Scambo para 20 idiomas usando next-intl, com slugs traduzidos e detecção automática de locale.

**Architecture:** next-intl v3+ com middleware de locale detection, mensagens em JSON (~355 strings/idioma), páginas movidas para `app/[locale]/`, API routes permanecem fora do segmento de locale. Slugs traduzidos via `pathnames` config.

**Tech Stack:** next-intl v3+, Next.js Middleware, JSON messages, TypeScript

- [ ] **Step 1.3: Create src/i18n/request.ts**

```ts
import { getRequestConfig } from 'next-intl/server';
import { routing } from './routing';

export default getRequestConfig(async ({ requestLocale }) => {
  let locale = await requestLocale;
  if (!locale || !routing.locales.includes(locale as any)) {
    locale = routing.defaultLocale;
  }
  return {
    locale,
    messages: (await import(`../messages/${locale}.json`)).default,
  };
});
```

### Task 7: Adicionar seletor de idioma no Header

**Files:**
- Modify: `src/components/Header.tsx`

- [ ] **Step 7.1: Add locale switcher to Header**

```tsx
import { useLocale, useTranslations } from 'next-intl';
import { usePathname, useRouter } from '@/i18n/routing';
import { routing } from '@/i18n/routing';
import { useTransition } from 'react';

export default function Header() {
  const t = useTranslations('header');
  const locale = useLocale();
  const router = useRouter();
  const pathname = usePathname();
  const [isPending, startTransition] = useTransition();

  function handleLocaleChange(value: string) {
    startTransition(() => {
      router.replace(pathname, { locale: value });
    });
  }

  return (
    <header>
      {/* existing header content */}
      <select
        value={locale}
        onChange={(e) => handleLocaleChange(e.target.value)}
        disabled={isPending}
        className="bg-gray-800 text-white text-sm rounded px-2 py-1"
      >
        {routing.locales.map((loc) => (
          <option key={loc} value={loc}>{loc.toUpperCase()}</option>
        ))}
      </select>
    </header>
  );
}
```
